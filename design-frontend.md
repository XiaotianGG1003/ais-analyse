# AIS 船舶轨迹分析系统 — 前端设计文档

## 1. 产品概述

AIS 船舶轨迹分析系统是一个现代化的 Web 应用，用于对 AIS（自动识别系统）船舶数据进行可视化展示与智能分析。系统面向海事监控人员、港口管理者和航运分析师，提供直观的轨迹回放、区域监控、距离计算和轨迹预测等核心功能。

## 2. 用户需求分析

### 2.1 核心用户角色
- **海事监控员**：实时查看船舶位置与轨迹，监控区域进出
- **航运分析师**：统计分析船舶航行数据（速度、距离、时间）
- **港口管理者**：掌握航道使用情况，评估交通密度

### 2.2 核心功能需求

| 功能模块 | 描述 | 优先级 |
|---------|------|-------|
| 船舶轨迹展示 | 在地图上展示船舶位置与历史轨迹 | P0 |
| 时间段查询 | 按时间范围筛选显示轨迹 | P0 |
| 航行统计 | 计算距离、时间、最大/平均速度 | P0 |
| 区域检测 | 判断船舶是否进入指定区域 | P1 |
| 两船距离 | 计算两艘船之间的距离 | P1 |
| 轨迹预测 | 基于历史数据预测未来轨迹 | P2 |

## 5. 界面设计规范

### 5.1 色彩体系

```
主色调（海事蓝）：  #0EA5E9（Sky-500）
辅助色：          #6366F1（Indigo-500）
成功色：          #10B981（Emerald-500）
警告色：          #F59E0B（Amber-500）
危险色：          #EF4444（Red-500）
背景深色：        #0F172A（Slate-900）
面板背景：        #1E293B（Slate-800）
卡片背景：        #334155（Slate-700）
文字主色：        #F1F5F9（Slate-100）
文字次要：        #94A3B8（Slate-400）
边框色：          #475569（Slate-600）
```

### 5.2 设计原则
- **深色主题**：减少视觉疲劳，适合长时间监控
- **信息密度适中**：关键数据一目了然，避免过载
- **层次分明**：通过阴影和透明度区分层级
- **动效克制**：使用平滑过渡，不使用花哨动画
- **响应式**：最小支持 1280px 宽度

### 5.3 组件规范
- **圆角**：小组件 6px，卡片 12px，面板 0px
- **阴影**：使用 `rgba(0,0,0,0.3)` 分层
- **字体**：系统默认等宽/无衬线字体，数据展示用等宽字体
- **间距**：基于 4px 网格系统（8/12/16/24/32px）
- **图标**：使用 Element Plus 内置图标 + 自定义海事图标

## 6. 交互设计

### 6.1 核心交互流程

#### 船舶轨迹查看
1. 用户在左侧面板输入 MMSI 或船名
2. 系统搜索并展示匹配船舶列表
3. 用户选择目标船舶
4. 地图自动定位并绘制轨迹
5. 右侧面板展示船舶详情与航行统计

#### 时间段查询
1. 用户选择时间范围（快捷/自定义）
2. 系统查询该时间段内的轨迹数据
3. 地图更新轨迹显示
4. 统计数据同步更新

#### 区域检测
1. 用户点击"绘制区域"按钮
2. 在地图上绘制矩形或多边形
3. 系统计算选中船舶是否进入该区域
4. 右侧面板展示检测结果

#### 两船距离
1. 用户选择两艘船舶
2. 系统计算两船间距离
3. 地图上绘制连线并标注距离
4. 右侧面板展示详细数据

#### 轨迹预测
1. 用户选择船舶并开启预测
2. 系统基于历史轨迹生成预测路线
3. 地图上用虚线绘制预测轨迹
4. 显示预测置信度

### 6.2 地图交互
- 鼠标滚轮缩放
- 拖拽平移
- 点击船舶弹出信息卡
- 双击定位放大
- 右键菜单（截图/测距）

## 9. API 对接

### 9.1 接口列表

| 接口 | 方法 | 路径 | 描述 |
|------|------|------|------|
| 搜索船舶 | GET | `/api/vessels/search` | 按 MMSI/船名搜索 |
| 船舶详情 | GET | `/api/vessels/{mmsi}` | 获取船舶基本信息 |
| 轨迹查询 | GET | `/api/vessels/{mmsi}/track` | 获取指定时间段轨迹 |
| 航行统计 | GET | `/api/vessels/{mmsi}/statistics` | 距离/速度/时间统计 |
| 区域检测 | POST | `/api/analysis/area-detection` | 检测是否进入区域 |
| 两船距离 | GET | `/api/analysis/distance` | 计算两船距离 |
| 轨迹预测 | GET | `/api/vessels/{mmsi}/prediction` | 轨迹预测 |
| 船舶列表 | GET | `/api/vessels` | 获取所有船舶列表 |
